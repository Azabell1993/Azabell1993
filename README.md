## Projects

### llmrc
Rust를 중심으로 구성한 로컬 LLM 실행 런타임 프로젝트입니다.  
양자화된 GGUF 모델을 로드하고, C++ 애플리케이션과 연동하여 추론 기능을 제공하는 구조를 다루고 있습니다.

#### 주요 내용
- Rust 기반 모델 로딩 및 런타임 관리  
- GGUF 포맷 모델 처리  
- FFI를 통한 C++ 연동  
- HTTP 인터페이스 형태의 추론 제공  
- macOS / Linux 기준 빌드 환경 구성

#### 목표
- 로컬 환경에서 재현 가능한 LLM 실행 구조 정립  
- 언어 간 역할 분리를 통한 확장 가능한 아키텍처 구성  
- 서비스로 연결 가능한 기반 런타임 확보

#### 상태
- 기본 실행 파이프라인과 구동 구조를 구현한 단계입니다.  
- 안정화 및 기능 확장을 지속적으로 진행하고 있습니다.
